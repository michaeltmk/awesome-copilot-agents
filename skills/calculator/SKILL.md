# Calculator

## Overview

This guide covers essential calculator operations using Typescript NodeJS libraries and command-line tools. It uses [big.js](https://github.com/MikeMcl/big.js/), described as:
> A small, fast JavaScript library for arbitrary-precision decimal arithmetic.

## Quick Start

```bash
cd scripts
npm ci || npm install
npm run build
npm run calculate "1 + 4.5 * (3-6) / 5"
```
